# Appendix I: SIMOPS & Permit Interface Card

**Status:** Draft on `draft/appendices`  
**Source:** MCR-028 (RPN 200) + Chapter 6 + Chapter 8  
**Purpose:** One-page field card for any job that has interfaces with other work parties, hot work, confined space, lifting, or simultaneous operations.

---

## I.1 When This Card Is Mandatory

Use this card whenever HPWJ will take place in the same area or time window as:
- Hot work (welding, grinding, cutting)
- Other high-pressure or abrasive blasting
- Confined space entry
- Lifting / crane operations
- Chemical cleaning or confined-space chemical work
- Any other contractor or discipline working nearby

**MCR-028 is RPN 200.** Poor SIMOPS coordination is one of the highest-risk failure modes.

---

## I.2 Daily SIMOPS Requirements (MCR-028)

| Requirement | Done | Notes |
|-------------|------|-------|
| Daily SIMOPS meeting held (or toolbox talk with all parties) | ☐ | |
| Named SIMOPS coordinator identified and present | ☐ | |
| All interfaces listed and risk-assessed | ☐ | |
| Physical zoning / barriers agreed and installed | ☐ | |
| Communication method between parties confirmed | ☐ | |
| Emergency stop / dump signals known to all parties | ☐ | |
| Hot-work controls (if any) verified and timed | ☐ | |
| Mist / water drift direction considered relative to ignition sources | ☐ | |
| Permit interfaces (PTW / LOTO / CSE) cross-checked | ☐ | |

---

## I.3 Hard Rules

- **No HPWJ** adjacent to hot work without physical separation **and** a named daily SIMOPS meeting.
- If communication between HPWJ team and other parties is lost → **stop work** until restored.
- Any change in other parties’ plans mid-shift requires an immediate re-assessment and re-brief.

---

## I.4 Quick Decision

Is there any other work party, hot work, or incompatible activity in the same area or time window?

- **No** → Proceed under normal controls.
- **Yes** → Complete this card + hold the daily SIMOPS meeting before pressurising. If the meeting cannot be held or zoning cannot be achieved → **do not start**.
