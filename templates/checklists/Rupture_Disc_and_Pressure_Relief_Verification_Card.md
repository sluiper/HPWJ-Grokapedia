# Rupture Disc & Pressure Relief Verification Card

**Status:** Live on main (merged after clean verification)  
**Linked to:** MCR-009, MCR-010, MCR-048 (directly tied to highest-RPN pre-use failures)  
**Purpose:** Dedicated one-page verification that the two most critical over-pressure protection devices are correct before every job.

---

## 1. Rupture Disc (MCR-009 + MCR-048)

| Check | Requirement | Pass | Fail | Notes |
|-------|-------------|------|------|-------|
| Correct rating | ≤ 1.2 × MAWP of the **lowest-rated component** in the system | ☐ | ☐ | |
| Single-use only | Never reuse a disc that has activated or been removed | ☐ | ☐ | |
| Spare on unit | Correct spare disc present and protected | ☐ | ☐ | |
| Tagging | Date of installation + rating clearly tagged | ☐ | ☐ | |
| Installation | Correct orientation, seating, and hold-down | ☐ | ☐ | |

**If any item fails → DO NOT PRESSURISE.**

---

## 2. Dump / Pressure Relief / Dead-Man System (MCR-010)

| Check | Requirement | Pass | Fail | Notes |
|-------|-------------|------|------|-------|
| Operator-controlled | Fail-safe / dead-man design | ☐ | ☐ | |
| Functional test | Operated and confirmed to dump pressure cleanly | ☐ | ☐ | |
| Accessibility | Operator can reach dump control from working position (or dedicated dump person when LOS lost) | ☐ | ☐ | |
| No bypass | No temporary ties, wedges, or defeat devices | ☐ | ☐ | |

---

## 3. After Any Plunger Change or Component Swap

- [ ] All ratings re-verified (MCR-015)
- [ ] New rupture disc calculation performed and correct disc fitted
- [ ] Dump system re-tested
