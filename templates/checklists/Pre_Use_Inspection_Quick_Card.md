# Pre-Use Inspection Quick Card (Companion to Full Checklist)

**Status:** Draft  
**Linked to:** Full Pre-Use Inspection Checklist + MCR-023 (RPN 210)  
**Purpose:** Pocket/laminated card for rapid confirmation that the critical items have been covered. Does **not** replace the full documented checklist.

---

## CRITICAL PRE-USE CHECKS (Must all be YES before permit)

### Pump
- [ ] Level, chocked, grounded, guards in place
- [ ] Rupture disc correct rating + spare on unit (MCR-009 / 048)
- [ ] Dump / dead-man functional and tested (MCR-010)
- [ ] Fluids, gauges, filters OK

### Hoses & Connections
- [ ] Visual reject criteria clear (no braid, bulging, cuts, kinks, weepage)
- [ ] Within life (flex 2 yr / standard 4 yr) + test records current
- [ ] Whip checks both sides of every HP connection (MCR-006)
- [ ] Correct couplings (prefer screw/face-seal where movement exists)

### Nozzles / Lances / Tooling
- [ ] Orifice clean & correct size; thrust direction verified
- [ ] AWD present, installed and verified on every flex lance (MCR-019/020)
- [ ] Reaction force calculated and inside all three additive controls (MCR-016)
- [ ] Nozzle within wear life

### People & Site
- [ ] Exclusion zone / barricading set (MCR-027 / 031 / 051)
- [ ] Minimum 3 competent persons (Anabeeb) + dump person if LOS lost (MCR-047)
- [ ] Medical Alert Cards carried by all (MCR-043)
- [ ] Communication method confirmed
- [ ] SIMOPS interfaces identified and coordinated if any (MCR-028)

### Final
- [ ] Full written checklist completed and signed by Inspector + Supervisor
- [ ] Any failed item rectified or equipment quarantined

**If any critical item is NO → DO NOT START. Quarantine and notify Supervisor / QHSSE.**

**This card is a memory aid only. The full Pre-Use Inspection Checklist is the controlled record.**
